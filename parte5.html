Parte 5 — Botones, Imágenes e Iconos

1. Botones

Bootstrap incluye un sistema completo de botones a través de la clase base .btn combinada con modificadores de color (.btn-primary, .btn-danger, etc.), tamaño (.btn-lg, .btn-sm) y estado (disabled, .active). También permite agrupar botones con .btn-group y construir barras de herramientas con .btn-toolbar.

2. Bootstrap Icons

Bootstrap Icons es una biblioteca de más de 2 000 iconos SVG que se integra mediante <i class="bi bi-{nombre}"></i>. Se escalan con las utilidades .fs-* y se colorean con .text-*. Para usarlos se enlaza su CDN o se instala el paquete npm.

3. Imágenes

Bootstrap provee tres utilidades principales para imágenes: .img-fluid para hacer imágenes responsivas, .img-thumbnail para añadir borde y padding, y el elemento <figure> con sus clases .figure, .figure-img y .figure-caption para imágenes con leyenda semántica.

.img-fluid — Imagen responsiva

La imagen nunca supera el ancho de su contenedor (max-width: 100%; height: auto).
