Introducción a Bootstrap

¿Qué es Bootstrap?


Bootstrap es un framework de código abierto que facilita la creación de interfaces web responsive y móviles.

¿Cómo lo integramos a nuestro proyecto?


  • CDN: Copiar enlaces oficiales en el HTML, no requiere descargar archivos.
  • Descarga local: Descargar archivos compilados y vincularlos manualmente al proyecto.
  • npm: Ejecutar npm install bootstrap en la terminal, se usa en proyectos con Node.js y bundlers.
  • Composer: Ejecutar composer require twbs/bootstrap para proyectos PHP.
  • Plantilla starter: Copiar el código base completo que ofrece la documentación oficial de Bootstrap.

Sistema de grillas

Containers, Rows y Columns


Los containers son el componente base del sistema de grillas. Son contenedores que centran y dan ancho fijo o fluido al contenido.


Tipos de containers

  • .container
  • .container-fluid
  • .container-sm
  • .container-md
  • .container-lg
  • .container-xl
  • .container-xxl

Las filas son contenedores horizontales que agrupan columnas.

  • .row
  • .row-col-6

Las columnas son los hijos directos de las rows. Dividen la fila en 12 partes iguales

  • Dentro del container van las filas (.row)
  • Dentro de las filas van las columnas (.col)
  • Los containers no se anidan

Breakpoints

Son los puntos de interrupción que definen el comportamiento responsive de la cuadrícula.

Se usan como sufijos en otras clases de Bootstrap.

Tipos de breakpoints:

  • sm
  • md
  • lg
  • xl
  • xxl

Gutters

Los gutters son los espacios (padding) entre filas y columnas.

Pueden tener el rango de 0 a 5, donde g-0 es 0px y g-5 es 48px.

Tipos de gutters:

  • .g-*
  • .gx-*
  • .gy-*

Offset

Crea espacios a la izquierda de las columnas sin necesidad de crear columnas vacías.

Sirve para crear diseños más flexibles y responsivos.

Se puede usar números del 0 al 12.

La suma de columnas + offset debe ser ≤ 12
