Parte 7: Contenidos Agrupados

Contenidos Agrupados

Los componentes de contenido agrupado permiten organizar información de forma jerárquica y visualmente atractiva. Incluyen acordeones para secciones colapsables, carruseles para sliders de imágenes y cards para presentar contenido en contenedores flexibles. Estos componentes son fundamentales en interfaces modernas para optimizar el espacio y mejorar la experiencia del usuario.

¿Qué es un Acordeón?

El acordeón es un componente que organiza contenido en secciones colapsables.

Estructura: .accordion como contenedor principal, .accordion-item para cada sección, .accordion-header con .accordion-button que contiene data-bs-toggle="collapse" y data-bs-target.

Variante: .accordion-flush elimina bordes y fondos para un estilo minimalista.

¿Qué es un Carrusel?

El carrusel es un slider que muestra contenido multimedia de forma cíclica.

Componentes: .carousel como contenedor, .carousel-inner con .carousel-item. Controles: .carousel-control-prev/next e indicadores .carousel-indicators.

Atributos: data-bs-ride="carousel" para autoplay, data-bs-interval="2000" para velocidad de 2 segundos, y .carousel-fade para transición suave.

¿Qué son las Cards?

Las cards son contenedores flexibles que agrupan información relacionada.

Estructura: .card como contenedor, .card-body para contenido, .card-title para títulos, .card-text para texto, .card-img-top para imágenes.

Variantes: .card-header y .card-footer para secciones adicionales. Layout horizontal combinando .row.g-0 con .col-*.
